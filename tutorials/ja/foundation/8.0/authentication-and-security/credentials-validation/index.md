## 概説
{: #overview }
**CredentialsValidationsSecurityCheck** は、リソースへのアクセスを認可するために任意の資格情報を検証する単純なフロー向けです。 設定されている試行回数に達した後にアクセスをブロックする組み込み機能も提供されます。
