## 概述
{: #overview }
**UserAuthenticationSecurityCheck** 类继承 **CredentialsValidationSecurityCheck** 的所有功能并添加：

- 创建用户身份的 `AuthenticatedUser` 概念。
- 可选的**记住我**功能
