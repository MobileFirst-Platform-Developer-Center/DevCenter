## 개요
{: #overview }
**CredentialsValidationsSecurityCheck**는 자원에 대한 액세스를 허용하기 위해 임의의 신임 정보를 유효성 검증하기 위한 단순 플로우를 위한 것입니다. 지정된 시도 횟수 후에 액세스를 차단하는 기존 제공 기능도 제공됩니다. 

학습서를 선택하십시오.
