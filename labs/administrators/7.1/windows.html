Overview

Follow the instructions below to learn how to install MobileFirst Platform Server on a Windows workstation.

For this purpose, Windows Server 2012 R2 64bit distribution will be used and together with DB2 Workgroup Server V10.5, WebSphere Application Server Liberty Core 8.5.5.5, MobileFirst Server including Operational Analytics and Application Center.

Pre-requisites:

  1. Either use a Windows 2012 R2 64bit workstation or a virtual machine. A trial image can be downloaded from Microsoft.com. Setup latest version of VMware Player / VMware Workstation / VMware Fusion or VirtualBox (depending on the disk image that you will select to download).
  2. Download product distributions.
    • Database – DB2_10.5.0.3_limited_Win_x86-64
    • MobileFirst Server – MFPF_7.1_INST_MGR_REP_MF_SVR_EN
    • WebSphere Application Server – WAS_LIBERTY_CORE_V8.5.5.5_MP_ML

    You can get these in three different ways:
    • If you are a customer and have a license – download them from Passport Advantage
    • If you are a Business Partner and have a Value Package – download them from Partner World
    • Request trial licenses from IBM representatives

Setup pre-requisited software

Description: Install additional apps and pre-requisites required to setup and run MobileFirst Platform on Windows Server.
Time to complete: 15 minutes

Setup IBM DB2 Workgroup Server for MFP 7.1 on Windows

Description: Setup DB2 Workgroup Server V10.5 that we will use as a database for MobileFirst Platform.
Time to complete: 12 minutes
Lab files:
  • IBM Installation Manager
  • Demo project on GitHub.com (.war, .adapter and .wlapp files can be found in the \bin folder)

Setup WAS Liberty Core on Windows

Description: Setup Installation Manager and then install WAS Liberty. Create server instances for MobileFirst Server and Analytics server.
Time to complete: 15 minutes

Setup MobileFirst Platform Server 7.1, Operational Analytics and Application Center on Windows

Description: Using Installation Manager install MobileFirst Platform Server. Setup Analytics server.
Time to complete: 15 minutes

Setup runtime and deploy Test application and adapter to verify MFP setup on Windows

Description: Using the Server Configuration Tool, create a MobileFirst Platform Server configuration and then setup the runtime environment. Deploy test application and adapter and verify server installation using the test application.
Time to complete: 25 minutes
